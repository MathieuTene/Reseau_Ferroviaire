# Projet Java / JavaFX — Simulateur de réseau ferroviaire

Ce dépôt est un **squelette** pour le TP (modèle + JavaFX) :
- `fr.eilco.projet.model` : classes métier (réseau, nœuds, segments, trains…)
- `fr.eilco.projet.sim` : moteur de simulation (tick, mise à jour)
- `fr.eilco.projet.ui` : rendu JavaFX (dessin du réseau, interactions simples)

## Prérequis
- JDK 17 (ou supérieur)
- Gradle (wrapper inclus)

## Lancer
- Linux/macOS: `./gradlew run`
- Windows: `gradlew.bat run`

## Ressources
- Placez vos images (si vous en utilisez) dans : `src/main/resources/images/`
